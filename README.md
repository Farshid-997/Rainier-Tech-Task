# Technology Used

1. Language- Typescript
2. Backend- Express JS
3. Database- Postgres
4. ORM -Prisma
5. Deploy- Vercel
6. Testing- Postman


# Features

1. User can Register 

2. User can Login,when login an  access token will be generated

3. Only Admin can add course, update course , delete course check the role based on the token
